Install ubuntu fresh 20.04LTS

logon dengan root akses pada ubuntu

  sudo su
  masukan password root ubuntu nya

lalu download file auto installer nya
  https://github.com/risky-budiman/acs-auto-install-fresh.git
dan setelah terdownload load file auto isntalaer nya

  sh ./acs_installer.sh

seletlah instalasi selesai load db backup nya.

  sh ./acs_db.sh
